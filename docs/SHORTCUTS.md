# Keyboard shortcuts

| Action | Shortcut |
| --- | --- |
| Add project | ⌘N |
| New workspace | ⇧⌘N |
| Search across projects (Quick Open) | ⇧⌘O |
| Scan for `.env` files | ⇧⌘F |
| Import `.envenc` | ⌘I |
| Save file (opens Save Review) | ⌘S |
| Settings | ⌘, |
| Remove selected project(s) | ⌫ |

Plus the native macOS window ones: ⌘T-style tab management via the Window menu
(every EnvHub tab is a full window — sidebar included), Merge All Windows, and
double-click a sidebar project for its own window (⌘-double-click for a tab).
